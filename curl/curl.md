# cURL

* Curl POST JSON
	* `curl -X POST https://reqbin.com/echo/post/json
   -H 'Content-Type: application/json'
   -d '{"login":"my_login","password":"my_password"}'`

* POST Body
	* `curl -X POST https://reqbin.com/echo/post/json 
   -H "Content-Type: application/json"
   -d '{"productId": 123456, "quantity": 100}'  `
